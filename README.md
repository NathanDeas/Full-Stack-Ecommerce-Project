# Full-Stack Ecommerce Project

An ecommerce web application built with a React frontend and ASP.NET Core backend.  
This project will demonstrates a full-stack workflow including product listing, cart management, and user authentication.

---

## Features
- Product catalog displayed on the homepage
- Shopping cart functionality(future)
- User account management(future)
- API built with ASP.NET Core and Entity Framework

---

## Tech Stack
**Frontend**
- React
- Vite
- CSS (custom styling)

**Backend**
- ASP.NET Core
- Entity Framework Core
- SQL Server

**Other**
- Git for version control
- npm for package management
